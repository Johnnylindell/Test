# Familjedashboard startpaket

En enkel startmall för barnfamiljer som vill få upp vardagen på en väggskärm.

## Innehåll

- `index.html` – fristående dashboard som kan öppnas i browsern
- `data.json` – exempeldata för vecka, rutiner och inköp
- `guide.md` – kort installationsguide

## Så använder du den

1. Öppna `index.html` i en browser.
2. Ändra texterna direkt i `data.json`.
3. Lägg filerna på en surfplatta, liten server eller Home Assistant-panel.

## För vem

För familjer som vill ha synlig veckovy, morgonrutin, kvällsrutin och inköpslista utan att bygga ett helt system från början.
